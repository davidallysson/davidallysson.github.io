About HTML

HTML is one of the languages we use to develop websites. It is considered the skeleton of the site. The acronym HTML stands for Hypertext Markup Language. Every HTML document has markers and it is with them that we establish structures within the page.

About CSS


CSS - Cascading Style Sheets - is a language used to define the appearance of elements defined with HTML. It is this language that defines all the design and all the beauty of your website, called "style". The biggest advantage is separating the format and content of a document.

About JavaScript


JavaScript is a programming language used to control HTML and CSS to manipulate behaviors on the page. These behaviors include events, animations and everything that can make the site more dynamic.

Close

About PHP


PHP is a language that allows you to create dynamic websites, allowing you to interact with the user through forms, URL parameters and links. The difference of PHP with regard to languages like Javascript is that the PHP code is executed on the server (server-side). In this way it is possible to interact with existing databases and applications on the server, with the advantage of not exposing the source code to the client. This is extremely useful when the program is dealing with passwords or any kind of confidential information.

SQL stands for structured query language, that is, a data management language that interacts with the database based on a relational model. It is with her that I search, change and save information from the site / system that I build. With this you can register all the products of your store.

About Ruby


Ruby is an interpreted, object-oriented programming language that is used on the server side - translating in other words: it is powerful, agile, and flexible. It is responsible for performing all the calculations and commands necessary to develop a system, a blog, a commercial website and others.

About Materialize


Materialize is a framework developed to facilitate and speed development when it comes to design. By integrating HTML, CSS and JS, it is possible to apply the modern "Material Design" - a design language developed by Google recently.

About Git


Git is a version control system for tracking changes in computer files and coordinating work on those files among multiple people. In a nutshell, with Git it is possible to work in groups on the same project and develop it in versions.
