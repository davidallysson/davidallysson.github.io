Afinal, o que é HTML?

HTML é uma das linguagens que utilizamos para desenvolver websites. É considerada o esqueleto do site. O acrônimo HTML vem do inglês e significa, em português, Linguagem de Marcação de Hipertexto. Todo documento HTML possui marcadores - do inglês: "tags" - e são com eles que estabelecemos estruturas dentro da página.

Afinal, o que é CSS?


O CSS - em português, Folhas de Estilos em Cascata - é uma linguagem utilizada para definir a aparência de elementos definidos com o HTML. É essa linguagem que define todo o design e toda a beleza do seu website, chamada de "estilo". Sua maior vantagem é efetuar a separação entre o formato e o conteúdo de um documento.

Afinal, o que é JavaScript?


JavaScript é uma linguagem de programação utilizada para controlar o HTML e o CSS para manipular comportamentos na página. Esses comportamentos englobam eventos, animações e tudo que pode tornar o site mais dinâmico.

Afinal, o que é PHP?


PHP é uma linguagem que permite criar websites dinâmicos, possibilitando uma interação com o usuário através de formulários, parâmetros da URL e links. A diferença de PHP com relação a linguagens como Javascript é que o código PHP é executado no servidor (server-side). Desta maneira é possível interagir com bancos de dados e aplicações existentes no servidor, com a vantagem de não expor o código fonte para o cliente. Isso é extremamente útil quando o programa está lidando com senhas ou qualquer tipo de informação confidencial.

Afinal, o que é SQL?


SQL significa, em Português, Linguagem de Consulta Estruturada, ou seja, é uma linguagem padrão de gerenciamento de dados que interage com os principais bancos de dados baseados no modelo relacional. É com ela que consulto, altero e salvo informações do site/sistema que construo. Com ela, por exemplo, é possível cadastrar todos os produtos de sua loja.

Afinal, o que é Ruby?


Ruby é uma linguagem de programação interpretada, orientada a objetos e é utilizada do lado servidor (server-side) - traduzindo em outras palavras: é poderosa, ágil e flexível. É responsável por realizar todos os cálculos e comandos necessários para se desenvolver um sistema, um blog, um site comercial e afins.

Afinal, o que é Materialize?


O Materialize é uma ferramenta (framework) desenvolvida para facilitar e agilizar o desenvolvimento quando o assunto é design. Integrando HTML, CSS e JS, é possível, com ele, aplicar no seu site o moderno "Material Design" - que é uma linguagem de Design desenvolvida pela Google recentemente.

Afinal, o que é Git?


Git é um sistema de controle de versão de arquivos. Através dele é possível desenvolver projetos no qual diversas pessoas podem contribuir simultaneamente, editando e criando novos arquivos e permitindo que eles possam existir sem o risco de suas alterações serem sobrescritas. Em poucas palavras, com o Git é possível trabalhar em grupo em um mesmo projeto e desenvolvê-lo em versões.
